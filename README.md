# Typing-speed-test
This is a cool website where you can test out your typing speed while maintaining your accuracy. If you make one small mistake, your accuracy will drop in real-time, so being accurate while maintaing a good speed is the goal here. The website's name is TypeFlow and the whole concept of this website is inspired by various different typing websites like Monkeytype, Nitro type, etc. but I wanted to make something different, where they user can't make a mistake or else accuracy/wpm will drop. 

# Background
I was inspired by typing test websites like Monkeytype, and such and I've been practicing my typing speed for years and I have seen an improvement in my typing speed by using websites like Monkeytype and I'm consistently clocking over 100 WPM now. So, I wanted to make a website of my own and publish it in hackclub.
